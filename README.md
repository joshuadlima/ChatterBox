# ChatterBox
An app to chat.. with anyone? everyone? 

### Prerequisites
- Docker
- Docker Compose
- Git

### How to use the app?
- Install the latest Android app release from releases.
- clone the repository and go to the ChatterBox\chatterbox_backend\chatterbox_django_app directory
- run docker-compose up
- the django backend + redis instance should be up and live on http://localhost:8000/ (or any tunneled resource)
